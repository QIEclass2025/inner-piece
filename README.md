# Inner-Piece : 마음 챙김 도구 (Web GUI)

기존의 CLI 기반 Inner-Piece 도구가 프리미엄 웹 애플리케이션으로 업데이트되었습니다.
이제 브라우저에서 더 편안하고 직관적인 인터페이스로 마음 챙김을 실천해보세요.

## ✨ 주요 기능
- **SOS 모드 (4-7-8 호흡)**: 
  - 시각적인 호흡 가이드 (원형 애니메이션)
  - "Breathe~", "멈춤", "내뱉기" 타이밍에 맞춘 코칭
  - 세션 종료 후 그라운딩(오감 확인) 기록 가능
- **ABCDE 사고 전환 훈련**: 
  - 단계별 마법사(Wizard) 인터페이스로 부담 없이 생각 정리
  - A(사건) -> B(믿음) -> C(결과) -> D(논박) -> E(효과) 흐름 유도
- **마음 일지 (History)**: 
  - 카드형 UI로 과거의 기록을 한눈에 확인
- **프리미엄 디자인**: 
  - 편안함을 주는 딥 블루 & 틸(Teal) 컬러 팔레트
  - Google Fonts (Outfit, Inter) 적용

## 🚀 실행 방법

### 1. 환경 설정
Python이 설치되어 있어야 합니다. 필요한 라이브러리를 설치하세요.
```bash
pip install flask
```

### 2. 서버 실행
프로젝트 폴더에서 아래 명령어를 실행합니다.
```bash
python app.py
```

### 3. 접속하기
브라우저를 열고 아래 주소로 접속하세요.
[http://127.0.0.1:5000](http://127.0.0.1:5000)

## 📂 프로젝트 구조
- `app.py`: Flask 웹 서버 메인 파일
- `models.py`: 데이터 모델 및 저장/불러오기 로직
- `templates/`: HTML 화면 파일들 (`index.html`, `sos.html` 등)
- `static/`: CSS 스타일시트 (`style.css`)
- `inner_peace_history.json`: 기록 데이터 저장 파일
